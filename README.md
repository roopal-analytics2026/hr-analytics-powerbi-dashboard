# hr-analytics-powerbi-dashboard
Power BI dashboard analyzing employee attrition, workforce trends, and key HR metrics using interactive visualizations and DAX.
📊 HR Analytics Dashboard – Power BI
📌 Overview :
This project focuses on analyzing employee data to understand attrition trends, workforce distribution, and key HR metrics.
📂 Dataset :
Employee dataset including age, department, salary, and attrition status
🛠 Tools Used :
Power BI
Power Query (Data Cleaning)
DAX (Calculated Measures)
📈 Key Features :
Attrition analysis by department and job role
Employee distribution by age and salary
KPIs: Attrition Rate, Total Employees, Average Salary
Interactive dashboard with slicers and filters
🔍 Key Insights :
Identified higher attrition in specific departments
Observed trends based on age group and salary range
Highlighted areas needing employee retention strategies
